# App Routes

## Public Routes
- `/`
  - Landing page
- `/login`
  - Supabase login
- `/signup`
  - Supabase signup
- `/jobs`
  - Public job listing page for candidates

## Protected Dashboard Routes
- `/dashboard/admin`
  - Admin overview
- `/dashboard/hr`
  - HR overview
- `/dashboard/employee`
  - Employee overview
- `/dashboard/candidate`
  - Candidate overview

## Recruitment Routes
- `/recruitment/jobs`
  - Job postings management
- `/recruitment/applications`
  - Candidate applications
- `/recruitment/interviews`
  - Interview scheduling and summaries
- `/recruitment/assessments`
  - Candidate assessments

## Employee Routes
- `/employees`
  - Employee directory
- `/employees/[id]`
  - Employee profile
- `/employee/requests`
  - Employee update requests

## Analytics Routes
- `/analytics`
  - HR analytics dashboard
- `/analytics/performance`
  - Performance metrics
- `/analytics/flight-risk`
  - Employee flight risk insights

## Chatbot Routes
- `/chatbot`
  - Role-aware HR chatbot

## Settings Routes
- `/settings/profile`
  - User profile settings
- `/settings/admin`
  - Admin settings

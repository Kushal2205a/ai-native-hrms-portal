# Project Scope

## Product
AI-native HRMS web application built with Next.js and Supabase.

## Core Roles
- Admin
- HR
- Employee
- Candidate

## Modules in Scope
1. Recruitment & Talent Acquisition
   - Candidate sourcing
   - Resume screening
   - Interview scheduling
   - Automated assessments
   - AI interview performance summary from transcript

2. Administrative HR Tasks
   - Employee record updates
   - HR communication
   - HR chatbot for employee queries
   - Interview scheduling support

3. HR Analytics & Workforce Planning
   - Performance tracking
   - Training recommendations
   - Employee flight risk scoring

## Out of Scope for Initial Build
- Payroll
- Tax calculation
- Benefits management
- Complex leave accrual
- Compliance automation
- Expense management

## Architecture
- Frontend: Next.js App Router
- Backend: Next.js Server Actions and Route Handlers
- Auth: Supabase Auth
- Database: Supabase PostgreSQL
- Storage: Supabase Storage
- UI: Tailwind CSS + shadcn/ui
- AI: Server-side LLM API calls only
